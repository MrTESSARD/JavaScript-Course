Exercices sur les numbers & Math

Exercice 1 :

  • Déclarez 4 variables avec des noms différents (sensiblements différents et représentatifs).
  • Ajouter les 2 premières variables et stocker le résultat dans une nouvelle variable
  • Multipliez le résultat des 2 dernières étapes et stocker le résultat dans une variable resultatFinal
  • Ecrivez un calcul qui permet de vérifier si le résultat final est un nombre pair. Expliquez pourquoi dans un commentaire in-line. Stockez ce calcul dans une variable pairImpair

Exercice 2 :

  • Cet exercice possède plusieurs énoncés à vérifier à l'aide de formules différentes
  • Vous possédez 3 groupes de données, à chaque fois un énoncé et 2 variables
  • Pour chacun de ces groupes, il faudra écrire un test qui prouve ou dément l'affirmationde l'énoncé
  • Ces tests seront à insérer dans des variables respectives qui s'appelleront
  • comparaisonPoids pour l'énonce 1
  • comparaisonTaille pour l'énonce 2
  • concordanceMdp pour l'énonce 3
