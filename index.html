ES6 Syntax

Avec les nouvelles mise à jour récurrentes de JavaScript ("officiellement" appellé ECMAScript) on observe l'apparition de nouvelles syntaxes, mais aussi de nouvelles methodes liées aux données qui existaient déja

La plupart de ces changement ayant pour but de faciliter la vie du développeur, il convient de les connaître afin de factoriser le code et le rend aussi plus agréable à écrire.
