Exerices sur les variables

Exercice 1 :

  • Déclarez une variable monNom
  • Initialisez la variable monNom (vous pouvez bien entendu utiliser le votre) sur une autre ligne
  • Déclarez une variable monAge et l'initiliser sur la même ligne

Exercice 2 :

  • Déclarez et initialisez une variable monPrenom qui contiendra un prénom (le votre est autorisé encore une fois)
  • Déclarez une nouvelle variable monNomComplet qui contiendra votre nom complet à partir des infos contenus dans monNom et monPrenom
  • Affichez ce nom complet dans la console

Exercice 3:

  • Corrigez le code de façon à ce que la console arrête d'afficher une erreur
